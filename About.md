# About ToDone

An offline-first task app that gets you from **To Do** — to **Done**.
Made by [Talentedunicorn](https://talentedunicorn.com) with 🫰 and [Svelte](https://svelte.dev).

---

## Features

- **Dual views** — switch between a paginated list and a drag-and-drop kanban board with one click
- **Live search & sort** — title search as you type, sort by updated or created, all at the database level
- **Markdown content** — every task supports rich markdown, rendered inline and in a full viewer
- **Undo delete** — deletions are toast-confirmed with a 5-second undo
- **Sync optional** — log in with Auth0 to sync tasks across devices
- **Export/Import** — all tasks as JSON

### Keyboard shortcuts

| Key      | Action                        |
| -------- | ----------------------------- |
| `n`      | New task                      |
| `/`      | Focus search                  |
| `v`      | Toggle list / kanban          |
| `Space`  | Toggle first task done / todo |
| `d`      | Delete first task             |
| `Escape` | Close editor or toast         |
